# ✅ To-Do List Web Application

🚀 A responsive task management application that helps users create, organize, update, and track daily tasks with an intuitive and productivity-focused interface.

## ✨ Features

* ➕ Add New Tasks
* ✏️ Edit Existing Tasks
* 🗑️ Delete Tasks
* ✅ Mark Tasks as Completed
* 📋 Organize Daily Activities
* 🔍 Search and Filter Tasks
* 📊 Track Task Progress
* 💾 Local Storage Support
* 🌙 Dark & Light Mode
* 📱 Fully Responsive Design

## 🛠️ Technologies Used

* 🌐 HTML5
* 🎨 CSS3
* ⚡ JavaScript (ES6)

## 🎯 Project Objective

The objective of this project is to provide users with an efficient task management solution that improves productivity and helps organize daily activities through a clean and interactive interface.

## 🚀 Skills Demonstrated

* Frontend Web Development
* DOM Manipulation
* Event Handling
* Local Storage Management
* Responsive Web Design
* UI/UX Development
* JavaScript Programming
